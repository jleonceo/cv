# CV · Juan Luis León Rodríguez

Currículum profesional en formato HTML/CSS, listo para imprimir en A4 o visualizar en navegador.

## Por qué un currículum vive aquí

Un currículum en documento cerrado se envía y se pierde de vista. Nadie sabe qué decía la versión de
hace seis meses, no se puede enlazar a una línea concreta y cada retoque genera otro fichero con
otro nombre.

Publicarlo como repositorio cambia tres cosas. Queda el historial completo, así que cualquier cambio
tiene fecha y motivo. Hay una dirección estable a la que apuntar. Y lo que afirma se puede contrastar
con el resto de la cuenta, que es donde está el trabajo que menciona.

A cambio se pierde la privacidad del envío: lo que se publica lo lee cualquiera, incluidos los datos
de contacto de más abajo.

## Contenido

`index.html`: un solo fichero con el texto, los estilos y los datos de los gráficos dentro. Depende
de dos recursos externos que se descargan al abrirlo, las tipografías de Google Fonts y la librería
[Chart.js](https://www.chartjs.org/) desde un CDN, así que **sin conexión el texto se lee entero y
los dos gráficos salen en blanco**.

## Cómo verlo

Abre `index.html` en cualquier navegador. Para guardar como PDF: `Ctrl+P` → "Guardar como PDF" →
tamaño A4.
